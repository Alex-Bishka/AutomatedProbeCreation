About

This viewer displays results from the SAE-guided automated probe data generation pipeline. Each run generates contrastive pairs (deceptive vs honest examples) and validates them using Goodfire's SAE features.

Features:

  • Browse all experimental runs
  • View SAE feature activations for each example
  • Inspect token-level activation heatmaps
  • Analyze quality scores and feature overlap
